Prac6
🔹 Step 1: Create Dataset in Excel
Steps:
Open Microsoft Excel
Create a new worksheet
Enter the following column headings in Row 1:
A1: Name
B1: Department
C1: Designation
D1: City
Enter employee data below headings (Row 2 onwards)
Ensure:
No empty rows/columns
Data is properly aligned

🔹 Step 2: Select Data Range
Steps:
Click on any cell inside the dataset
Press Ctrl + A (or manually select full table)
Make sure entire dataset is highlighted

🔹 Step 3: Create Pivot Table
Steps:
Go to Insert Tab
Click on Pivot Table
A dialog box appears:
Table/Range → already selected
Choose New Worksheet
Click OK
👉 A blank Pivot Table and Pivot Fields pane will appear

🔹 Step 4: Arrange Pivot Fields
Steps:
In Pivot Fields panel:
Drag Department → Rows area
Drag Name → Values area
By default:
It shows Count of Name (employees)
Optional:
Drag City → Columns area (to compare cities)
Drag Designation → Filters (to filter data)

🔹 Step 5: Perform Calculations
Steps:
In Values section:
Click drop-down → Value Field Settings
Select required function:
Count → Number of employees
Sum → Total values
Average → Average
Click OK
🔹 Step 6: Format Pivot Table
Steps:
Click anywhere inside Pivot Table
Go to Design Tab
Apply:
Table styles
Banded rows
Adjust column widths if needed

🔹 Step 7: Create Pivot Chart
Steps:
Click inside Pivot Table
Go to Insert Tab
Click Pivot Chart
Select:
Chart Type → Column Chart
Click OK
Result:
Chart shows:
X-axis → Department
Y-axis → Number of Employees

Step 8: Import Data from Data Warehouse (SSMS)
Steps:
Open SQL Server Management Studio (SSMS)
Connect to database
Click New Query
Enter SQL query:
SELECT * FROM Employee;
Click Execute

🔹 Step 9: Export Data as CSV
Steps:
In Result panel:
Right-click → Save Results As
Choose:
File type → CSV
Save file to your system

🔹 Step 10: Open CSV in Excel
Steps:
Go to saved location
Double-click CSV file
It will open in Excel
Use this data to:
Create Pivot Table
Perform analysis
